<context>
Current gsd-db.ts uses better-sqlite3/node:sqlite directly. All operations (open, close, query, insert, update, delete) must be abstracted.

Key operations to abstract:
- openDatabase(path) → boolean
- closeDatabase() → void
- isOpen() → boolean
- query(sql, params?) → any[]
- run(sql, params?) → { changes: number, lastInsertRowid: number }
- transaction(fn) → fn
- migrate() → void (schema initialization)
- getDbPath(basePath) → string

PREFERENCES.md location: `~/.gsd/PREFERENCES.md` (global) or `.gsd/PREFERENCES.md` (project)
Config field: `storage_backend: "sqlite" | "markdown"` (default: "sqlite")
</context>
